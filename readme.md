## APIs

- #### Get the quiz db file with the specified category from the back-end.

```
  GET /api/v1/quiz
```

- #### Get the status of the backend API.

```
  GET /api/v1/status
```

- #### Get the most updated version of the quiz json file.

```
  GET /api/v1/version
```
